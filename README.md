### Для сборки проекта импортировать библиотеки
```shell
npm install leaflet react-leaflet
```
Данные для карт принимаются в формате GeoJson их можно конвертировать с поддержкой русского языка на [сайте](https://kml-to-geojson.vercel.app/)
Далее перенести данные в файл ```components/lines/lines.js```

### Запуск
```shell
npm gatsby develop
```
